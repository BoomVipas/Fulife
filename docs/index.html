About The Game

Fulife is a simulation game where you can experience different career paths and life choices. Talk to NPCs, make decisions, and see where your virtual life takes you!

Installation Instructions (for downloaded version)

  1. Download and extract the game package
  2. Make sure you have Python 3.8+ installed
  3. Install Pygame: pip install pygame
  4. Run the game: python main.py

Game Controls

  • WASD or Arrow Keys: Move your character
  • E: Interact with NPCs and objects
  • I: Open inventory
