Assets Folder for FULIFE

This folder contains all the assets needed for the FULIFE game, including:

1. Images (in the images/ folder):
   - map.png - Game background map
   - player_sprite.png - Player character sprite sheet
   - CEO_fullbody.png - CEO character full body image
   - CEO_face.svg - CEO face for dialog boxes
   
2. Sounds (in the sounds/ folder):
   - background_music.mp3 - Background music
   - dialog_sound.mp3 - Dialog sound effects
   - walk_sound.mp3 - Walking sound effects
   - interact_sound.mp3 - Interaction sound effects

To use these assets in the game, you need to copy them to the corresponding folders in the fulife/ directory:
- assets/images/* → fulife/images/
- assets/sounds/* → fulife/sounds/
